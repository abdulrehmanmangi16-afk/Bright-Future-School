## 💾 DATA STORAGE

### Todo App
- **Storage Type:** Browser localStorage
- **Capacity:** ~5-10MB per domain
- **Persistence:** Survives browser restart
- **Clearing:** Clears when you clear browser cache
- **Backup:** Export to JSON file

### Joke Generator
- **Favorites:** Stored locally
- **History:** Last 20 jokes stored
- **Stats:** Tracked locally
- **Persistence:** Survives browser restart
- **Clearing:** Clears when you clear browser cache

## 🔄 SYNC ACROSS DEVICES

⚠️ **Important Note:**
- Each device has its own storage
- Todo tasks don't sync between devices
- Joke favorites separate per device
- **Solution:** Export todos and import on another device

## ❓ FAQ

**Q: Will my tasks/favorites be saved?**
A: Yes! They're saved in your browser's localStorage and persist even after closing.

**Q: Can I access from another device?**
A: No, localStorage is device-specific. Export your tasks to transfer.

**Q: What happens if I clear browser cache?**
A: Data will be deleted. Keep backups by exporting!

**Q: Do these apps need internet?**
A: Todo App = No. Joke Generator = Yes (for API calls).

**Q: Can I customize the apps?**
A: Yes! Fork the GitHub repo and modify the code.

**Q: How many tasks/jokes can I store?**
A: ~5-10MB worth (depends on device), typically 500+ tasks or 1000+ jokes.
